Bestandteile eines Packages

Ein "Package" eines Projekts besteht aus:

  1. 4 Preview Bilder ( + 1 Splash/Logoscreen)
  2. 1 Thumbnail
  3. Einer Beschreibung (in HTML)
  4. eine Beschreibung von Anforderungen
  5. 1 Button Bild
  6. [Das Programm selbst zum Download]

Nun die Obengenannten Punkte genauer:

  1. Die 3 Preview Bilder und das Splashbild werden für den Slider gebraucht. Das Splashbild ist immer das erste im Slider.
  2. Das Thumbnail ist für die Zufällige generierte "Kacheln" es steht in Verbindung mit der Kurzbeschreibung (Wenn Maus darüberfährt, wird die Kurzbeschreibung angezeigt
  3. Eine ausführliche Beschreibung mit Namen der App / Webpage auch in HTML geschrieben und wird auch mit PHP reingeladen.
  4. Ist eine Beschreibung der Anforderungen bzw. welches Wissen ich besitzen musste um diese App / Webpage zu schreiben. Besteht aus 3 - 5 Punkten.
  5. Ein Button welcher einen zur App / Webpage weiterleitet. Ist die App in keinem Store so ist dieser als Downloadbutton zu sehen, Webseiten als Link zur Seite.
  6. Als Binary in einem .zip Ordner gespeichert.

Ordnerstruktur:

  • <Projektname>
    • <slices>
      • <img>splash.png</img>
      • <img>1.png</img>
      • <img>2.png</img>
      • <img>3.png</img>
      </slices>
    • <thumbnail>
      • <img>thumb.png</img>
      </thumbnail>
    • <item>description.html</item>
    • <item>dependencies.html</item>
  • </Projectname>
